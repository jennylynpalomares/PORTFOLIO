Automated Follow-Up Reminders (Google Apps Script)

A lightweight automation that scans a billing sheet daily and emails a follow-up digest for any account crossing the 45-day threshold — removing the need to manually re-check the aging report every morning.

The Problem

Aging reports are only useful if someone checks them consistently. In a manual workflow, accounts quietly slip past the 45-day follow-up window because no one caught it that day.

How It Works

  • 1 Script runs on a daily time-based trigger inside Google Sheets
  • 2 Scans the aging column for any account crossing 45+ days outstanding
  • 3 Builds a short digest list of accounts needing same-day follow-up
  • 4 Emails the digest automatically — no one has to remember to check

Core Script

// Runs daily via time-based trigger
function sendFollowUpDigest() {
  const sheet = SpreadsheetApp.getActiveSheet();
  const data = sheet.getDataRange().getValues();
  const dueToday = [];

  data.forEach(function(row, i) {
    if (i === 0) return; // skip header row
    const daysOutstanding = row[7];   // column H
    const status = row[12];          // column M
    if (daysOutstanding > 45 && status !== 'Resolved') {
      dueToday.push(row[0] + ' — ' + daysOutstanding + ' days');
    }
  });

  if (dueToday.length > 0) {
    MailApp.sendEmail(
      'billing@example.com',
      'Follow-Up Digest — ' + dueToday.length + ' accounts due today',
      dueToday.join('\n')
    );
  }
}
✓ Result: zero manually-missed follow-ups across the sample sheet during testing — the digest surfaces overdue accounts automatically every morning.
